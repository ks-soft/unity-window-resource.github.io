デフォルト値の変更方法¶

様々なデフォルト値を、KsSoft/Plugins/KsSoftConfig.cs内で定義しています。

デフォルト値を変更したいときは、このファイルを修正してください。

アセットバンドル系¶

string AssetbundlePath¶

アセットバンドルの出力フォルダを変更

string AssetbundleExt¶

アセットバンドルファイルの拡張子

string ResourcePath¶

リソースデータとして出力するときに使用するパス

string deugPath¶

“http://“経由ではなく、”file://“経由でアセットバンドルを読み込むときに使用される。

Note

実機では無効の値です。

ウィンドウリソース系¶

uint WindowAssetbundleId¶

アセットバンドル経由で読み込むときにデフォルトでダウンロードするウィンドウアセットバンドルID

uint DefaultFontKind¶

フォントIDが指定されていないときに使われるデフォルト値

uint DefaultTextureId¶

テクスチャIDが指定されていないときに使われるデフォルト値

フォント系¶

uint FontsAssetbundleId¶

フォント群をアセットバンドル化する際に使用されるアセットバンドルID

メッセージデータシート系¶

Dictionary<e_Locale, uint> MessageDataId¶

ロケールデータを追加したときは、以下を参考に追加してください。 値は、対応するロケールをアセットバンドル化するときに使われるアセットバンドルIDです。

static private Dictionary<e_Locale, uint> m_dicStrMessageDataMulId = new Dictionary<e_Locale, uint>() {
  {e_Locale.JP, MulId.Id (0,2,0)},
  {e_Locale.EN, MulId.Id (0,2,1)},
};

アドレス/IP系¶

string httpserver = "http://xxx.xxx.xxx.xxx/"¶

HTTP経由でアセットバンドルをダウンロードするときのアドレスを指定してください。
