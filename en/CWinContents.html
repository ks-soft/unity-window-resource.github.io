CWinContents¶

CWinContents has a role as a container for holding the various controls .

CWinContents has been used in the following part .

  • CWindowBase has one of CWinContents. Thereby , window has some control .

  • CWinCtrlBase have it(it called content. Please refer to here .

  • CWinCtrlBase has it in a list format. we called the content list.

The first two are defined structure by the window script.

The third(content list) with a list by copying the content with the control as a template .

Function/Method¶

WinCtrl find<WinCtrl>(uint ctrlId)¶
Parameters uint ctrlId:
 Control ID

It Find the control of the relevant ID. If the object does not exist in the content , recursively Find in the content of the control.However , in the content list does not take into search. Thus , control is determined always unique.

int getIndex(uint ctrlId)¶
Parameters uint ctrlId:
 Control ID

It looks for whether the specified control ID is stored in what number. If not found , it returns -1 . We do not want to search recursively Unlike the 「find」.

Index value has been returned to be used as an argument of this [N]. Thereby enabling high-speed access.

CWinCtrlBase this[N]

You can use the index value obtained by 「 getIndex 」.

int count¶

It returns the number of controls in the content.

ClipRect::State clipState¶

This is effective only content , which is stored in the content list (ex: LISTBOX , LISTBOXEX ).

ClipRect.State.Inside

Completely within the region

ClipRect.State.Clipped

Clipping necessary

ClipRect.State.Outside

Rendering unnecessary (completely outside the region)

float width¶

Get the width of the content. It is automatically calculated from the size of the control.

float height¶

Get the height of the content. It is automatically calculated from the size of the control.

Vector2 size¶

Get the size of the content. It is automatically calculated from the size of the control.

Vector3 position¶

Display position of content

CWinCtrlBase parent¶

Get a control’s parent.

Content with the CWindowBase returns a null.
