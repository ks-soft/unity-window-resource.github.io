CTextureResourceMgr¶

Call AddComponent in Awake of CMainSystem.

This is intended to manage the texture resources .

It is a container of CTextureResource.

Download or load from resource the CTextureResource. Already it returns the cache When it is loaded.

Window system also load a texture via this manager .

Resource creation method please refer to here .

Note

To read via asset bundle , it is necessary to register the CAssetBundleMgr .

CTextureResourceMgr¶

CTextureResource reference(uint id)¶
Parameters uint id:
 Asset bundle ID

It returns the texture resources . If when not cached and starts downloading .

When the texture resource exists even if not already downloaded , it will return the CTextureResource.

It is whether during the download you examine the CTextureResource.isLoaded.

When true, it is enabled .

If you can not download as an asset bundle, try either load as resource data .

If it still does not exist , it returns a null.
